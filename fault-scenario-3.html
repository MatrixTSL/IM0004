Fault Scenario 3: Bad Flow

Scenario: Bad Flow
Process: Flow sensor monitoring and control
Shift: Production shift during normal operation
Urgency: Medium - System performance affected
Difficulty: Beginner
Time: 20-30 minutes

Scenario Background

You're monitoring a flow system where the flow manometer gauge shows the system is delivering approximately 50 litres per minute too much flow.

The flow sensor on the I/O screen has an offset applied that is causing the system to deliver excessive flow rates. This is affecting system performance and efficiency.

The flow readings are consistently higher than the expected values, and the system is not operating within normal parameters.

Industry Context: This kind of problem tends to occur in industry because flow issues can be caused by blockages, airlocks in the piping, pump cavitation, or partially closed valves. Wear and tear on pumps or leaks in the system can also reduce flow, while buildup of debris in filters or strainers can create unexpected restrictions. If these issues aren't identified and fixed, the system struggles to maintain proper flow rates, leading to unstable operation.

Initial System Status

Expected Flow Rate: 100 L/min
Actual Flow Rate: 150 L/min

Troubleshooting Tips

  • Check coolant system maintenance schedules and filter conditions
  • Monitor thermal cycling patterns and system efficiency trends
  • Investigate the correlation between operational changes and performance
  • Use systematic approach: check flow, pressure, temperature, and quality
